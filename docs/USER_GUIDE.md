# User Guide

## Recommended Clients

- Windows: `v2rayN`
- Android: `v2rayNG`
- iPhone / iPad / macOS: `Streisand`
- macOS alternative: `FoXray`

## How Users Connect

1. Copy the `subscription URL`, `VLESS` link, or scan the `QR` from the admin panel.
2. Import it into a supported Xray client.
3. Select the profile and connect.

## Important Notes

- This project uses `VLESS + REALITY`, not a classic client-certificate VPN workflow.
- If a client application does not support `REALITY` or `XTLS Vision`, upgrade the app or switch to a newer client.
- Treat subscription URLs as credentials.
