# Диагностика

## Панель Открывается С Предупреждением О Сертификате

Если панель работает по IP и `PANEL_TLS_MODE=internal`, браузер будет предупреждать о сертификате. Для публично доверенного сертификата панели нужен домен.

## Повторный Browser Basic Auth Prompt

Если браузер продолжает показывать старый auth popup после изменения Caddy-конфига, сделай полный reload страницы или открой панель в новой вкладке.

## Админ Не Может Войти

- проверь пароль
- проверь, не требуется ли `TOTP 2FA`
- проверь rate limiting на auth endpoints
- проверь логи `API` и `Caddy`

## Клиент Не Подключается

- убедись, что используемый клиент поддерживает `REALITY`
- проверь, что `XRAY_REALITY_PUBLIC_KEY`, `shortId` и `serverName` совпадают с выдаваемой подпиской
- проверь, что клиент не истёк и не заблокирован политикой лимитов

## Backup / Restore

- перед restore всегда запускай `--dry-run`
- не выполняй restore без абсолютного пути к архиву
- после restore проверь `healthz`, `readyz` и live sync с `Xray`
